FuzzyAutoComplete 设置指南

设置

打开设置

Editor -> FuzzyAutoComplete -> Plugin settings...

设置内容

Basic

Sort by Score

分数越高越精确
如果选中,那么匹配结果按照分数(这个分数是内部计算的分数)从高到低排列
如果没有选中,那么匹配结果按照下一个字母A-Z的顺序排列

Filter by

threshold取值在0.01~1之间,值越大,匹配的内容就越精确

static threshold (静态临界值)

自动匹配列表中的数据只有分数大于该值的情况下才会显示,否则永不显示

dynamic threshold
动态阈值
意义等同于静态阈值,区别在于:当匹配列表中的数据低于该数值时依然能够显示,并且只显示最优匹配结果

Require Prefix

如果选定,匹配值则会必须包含指定长度的前缀,这样会缩小范围,更加精确,个人视情况选择 (从输入值开始匹配,不会从中间开始匹配了)

Show Inline Preview

在书写的地方会有灰色文字提示,显示了完整的方法名称

Show List Header

在列表上方显示正在书写的内容

Advanced

Include Timings in Header View

在list Header上显示自动匹配所耗费的时间

Show Scores

显示每项匹配的分数

Parallel Scoring

Experimental

Correct Letter Case

自动改变大小写

Correct Word Order

自动改变文字顺序
