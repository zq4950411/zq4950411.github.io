段落和换行

和LATEX一样，用一个以上的空行代表分段，而没有空行的换行被忽略，所以允许段内强制换行
单个回车，视为空格；连续回车，才能分段。回车前在行尾加两个空格，即可段内换行。

Q:在 Markdown 文档里如何打出空行？
A:直接使用 html 标签，即 <br />
Q:使文本保持原样输出
A:在开头输入两个以上空格
