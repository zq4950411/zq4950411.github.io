Hexo使用--基础篇

发表于 2015-12-25   |     |   阅读次数

简介

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

安装

前期准备

请先确保已经安装了Node.js和Git

可以使用以下命令检测是否已安装：

如果检测出来没有安装，那么就点击上面的超链接进行安装

Node.js推荐使用nvm安装

安装Hexo

执行命令开始安装

$ npm install -g hexo-cli

创建站点文件夹

在你比较方便的目录创建一个文件夹，并在这个文件夹下执行以下命令：

$ hexo init <folder>
$ cd <folder>
$ npm install

写作

笔者推荐使用Markdown语法进行写作，如果读者对Markdown语法不太清楚，可以查看此处

使用下列命令来创建一篇新文章

$ hexo new "hello-world"

这时你可以打开站点文件夹下/source/_posts内的文章进行修改，修改完成后记得保存。

预览

Hexo使用可以本地创建服务来对即将生成的文件进行预览。

$ hexo server

这时候你就可以在浏览器中打开http://localhost:4000/对网页进行预览。

生成静态文件

Hexo中如果要想发布博客到服务器供别人阅读，就必须先生成html形式的静态文件。

$ hexo generate

生成的静态文件默认存放在站点文件夹下的/public文件夹下。这里的静态文件可以拷贝到任意服务器单独运行。
