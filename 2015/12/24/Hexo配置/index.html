网站配置

参数 描述 参考值
title 网站标题 思路人
subtitle 网站副标题 有时候一点点就是一大步
description 网站描述 人生如戏，全靠心细
author 作者 翟强
language 网站使用的语言 zh-Hans
timezone 网站时区。一般不用设置

读者可以参考本站的具体显示位置来明确每个值的作用。

其他配置

笔者本来想把所有参数都讲解一遍，仔细思考之后发现很多参数保持默认设置就好。这里只摘要出笔者认为比较好的一些参数进行讲解。

参数 描述 默认值
auto_spacing 在中文和英文之间加入空格 false
external_link 在新标签中打开链接 true
per_page 每页显示的文章量 (0 = 关闭分页功能) 10
theme 当前主题名称。值为false时禁用主题
